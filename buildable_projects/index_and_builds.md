# Build Instructions
## CyberChef
(Requires node v11.15.0, npm, and Grunt)
1. `cd CyberChef/` 
2.`npm install`
3. `grunt prod`

CyberChef is built and the output is in *build/prod/*. Unzip the CyberChef\_v9.x.x.zip archive wherever you need it to live. Make a browser bookmark for index.html.
